# Sauce Demo – QA Portfolio Project

## Overview
This repository contains a self-directed manual QA testing project for the Sauce Demo web application. The project demonstrates a full QA test cycle, including test planning, test case design, execution, defect reporting, exploratory testing, and test reporting.

---

## Application Under Test
- Name: Sauce Demo
- URL: https://www.saucedemo.com
- Type: Web-based e-commerce demo application

---

## Testing Scope
- Login functionality
- Product listing and sorting
- Cart functionality
- Checkout process
- Logout and navigation

---

## Test Artifacts
- **Test Plan** – Defined scope, approach, risks, and deliverables
- **Test Cases** – Manual test cases covering core user journeys
- **Test Execution Results** – Summary of executed tests and outcomes
- **Bug Reports** – Documented defects with reproduction steps and evidence
- **Exploratory Testing Notes** – Findings from unscripted testing sessions
- **Trello Workflow Documentation** – Task and defect tracking approach

---

## Tools Used
- Trello (test tracking and defect management)
- GitHub (documentation and version control)
- Google Chrome (browser testing)

---

## Summary
All planned test cases were executed successfully. One low-severity UI defect was identified and documented. This project demonstrates practical, real-world QA testing skills suitable for junior or entry-level QA roles.
